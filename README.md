## 주요 기능 설명
1. 주변 상점 정보 불러 오기 기능
   - 사용자 기기 위치 값을 얻어 주변 상점 정보를 불러 옵니다.
   - 사용자에게 리스트 형식으로 불러온 정보를 제공 합니다.
   - 
2. 마커 기능 및 재고

   - 마커를 클릭하면 재고 정보를 표시합니다.
   -
